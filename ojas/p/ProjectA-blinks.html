BLINKS

This project provides an alternate way for physically incapable people to communicate with the world without the use of any special language so that it makes them feel a bit more confident while making a conversation. It is basically for people suffering from locked-in syndrome, motor neuron disease, and ALS. It detects the opening and closing of eyelids of the patient using face recognition with the help of python and open CV. When only the right eye would be closed it would be detected as a dot and if only the left eye is closed it would be detected as a dash. Using this combination of dot and dash with the help of Morse code we would inbuild some set of commands like “H” for hunger, “T” for thirsty.

    OBJECTIVES :

  • To make a basic level of interaction possibilities for the customer.
  • It helps the customer to call out to others in case of an emergency
